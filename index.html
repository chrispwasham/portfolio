Accessibility

For the better part of a year in 2019-20, I worked on a project to improve the accessibility profile of a large corporate website. Following Web Content Accessibility Guidelines (WCAG), WAI-ARIA and published and tested design patterns, I enhanced page structures; complex tools like a navigation dashboard, interactive forms and selectors; dynamically changing content with moving focus; and numerous interactive page components.

I ensured that these enhancements provided comprehensive keyboard control (with visual indicators of keyboard focus), logical progression in keeping with the visual layout, as well as proper structure, semantics and supplemental audio support for the visually impaired.

In this project, I learned how to think through the user experience from different points of view, and using tools and browser plug-ins to report accessibility errors and test and validate solutions.
